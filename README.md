# Movie Apps

## Tech Stack
- **Flutter**
- **GetX** (State Management & Routing)
- **Hive** (Local Storage)
- **Build Runner**

## Features
- List movie
- Detail movie
- Favorite movie (local storage dengan Hive)
- State management menggunakan GetX
- Clean & simple UI
